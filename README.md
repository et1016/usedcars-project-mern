<h2>功能介紹</h2>

- 提供使用者註冊及登入
- 管理員帳號可以建立車籍資料
- 將商品加入至購物車
- 購物車結帳頁面
- 將購物車資料儲存至 localStorage
- 提供使用者搜尋車輛
- Google Map 嵌入

<h2>使用技術</h2>

- JavaScript
- React.js
- Redux
- Axios
- Node.js
- Express.js
- MongoDB
- VSCode

<h2>作品說明</h2>

- 使用 Redux 狀態管理工具建立購物車
- 使用 RWD 響應至不同裝置上
- 使用 Material-UI 和 SCSS 設計樣式
- 註冊密碼經加密處理
- 使用 MongoDB Cloud 儲存使用者帳號密碼
- 使用 GitHub Pages 進行部署，託管伺服器則使用 cyclic
